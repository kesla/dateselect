Awesome

a time-based job scheduler with CSS-like selectors.

About

Awesome is a time-based job scheduler - just like cron. With the help of CSS-like selectors, Awesome is easy to use while still being really powerfull. Node.js only.

Examples

With awesome, you can for example schedule a task to be run once every minute in february:

awesome("month.2 minute", function(){
		      doSomethingFunky();
		    });

Or create an alam clock:

awesome("day.weekday hour.7am minute.30", function(){
        console.log("7:30 is way to early to wake up.");
        alarmClock();
      });

We can also use '>' in the above example:

awesome("day.weekday > .7am > .30", function(){
        console.log("7:30 is way to early to wake up.");
        alarmClock();
      });
