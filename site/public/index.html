Dateselect

Schedule with CSS-like selectors.

About

Dateselect is a time-based job scheduler (like cron). With the help of CSS-like selectors, dateselect is easy to use while still being powerfull. Node.js only.

Examples

With dateselect, you can for example schedule a task to be run once every minute in february:

dateselect("month.2 minute", function(){
    doSomethingFunky();
});

Or create an alam clock:

dateselect("day.weekday hour.7am minute.30", function(){
    console.log("Wake up!");
    startTheRadio();
});

We can also use '>' in the above example:

dateselect("day.weekday > .7am > .30", function(){
    console.log("Wake up!");
    startTheRadio();
});
